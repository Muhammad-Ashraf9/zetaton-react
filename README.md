# Zetaton + React

## project setup

1.  clone the repo

```bash
git clone https://github.com/Muhammad-Ashraf9/zetaton-react.git
```

2.  cd into the project

```bash
cd zetaton-react
```

3.  install dependencies

```bash
npm install
```

4.  start the project

```bash
npm run dev
```
